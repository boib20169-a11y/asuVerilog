This project is about creating a functioning ASU that is capable of subtracting/adding any two 4 bit numbers. A ASU, 
seven segment display was created to implement the result in the Basys 3. Multiple test benches were developed for the
waveforms in debugging some of the issues I had with making the ASU work. The latest version of Vivado was used in order
to implement the files. The following design and simulation source file is located in the directory: ASU.srcs. 
With the preview of the waveforms as a png located at: ASUwaveformspreview. The demo for the board is located in
this drive : https://drive.google.com/file/d/1ETVzIjJNhh1JDhYfV6lYhkewicIvQdtk/view?usp=drive_link .
